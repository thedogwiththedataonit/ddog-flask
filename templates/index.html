Native Cloud Tools

Let's see how Datadog performs against AWS Cloudwatch, Azure Monitor, and GCP Monitor!


AWS

CloudWatch

Main features include a customizable dashboard, alarms, log management, metrics monitoring, servicelens, and synthetics.
CloudWatch is closely integrated with the AWS products, offering a large array of system level metrics and logs that can be correlated with the underlying credit or pricing of products.

Azure

Monitor

Main features include metrics, logs, and alerts/actions. Ontop of this, Azure offers many customized products that go hand in hand with Azure's monitoring service. Azure monitor is also the closest to Datadog's platform aesthetically.

GCP

Monitor

Main features include dashboards, metrics, alerting, and uptime checks. Also is able to monitor AWS products.
